# Weather Report Web App

A **web-based weather report application** built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the **Open-Meteo API**. Users can search for any city and get current temperature, weather conditions, wind information, and hourly forecast for the last 24 hours.

---

## Features
- Search for any city and get real-time weather updates.
- Displays **current temperature**, min/max temperature, weather conditions, sunrise/sunset times, and wind info.
- Shows **hourly temperatures** for the last 24 hours.
- Handles invalid inputs and provides error messages.
- Responsive and user-friendly design for desktop and mobile screens.
- Default city displayed on load (Bhubaneswar).

---

## Technologies Used
- **HTML & CSS**: For layout, styling, and responsive design.
- **JavaScript**: Handles API requests, user input, and dynamic content update.
- **Open-Meteo API**: Fetches accurate real-time weather data.
